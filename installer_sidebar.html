What will be created

  • All required sheets (SETTINGS, CLIENT DATA, PRODUCTS, ORDER_FORM_1 …)
  • Named ranges (ADMIN_LOGIN, section names, colours, variation groups)
  • Default values you can customise right away
  • The Order System menu in your spreadsheet toolbar
2

Run Setup

Click the button below. This takes about 15–30 seconds.

3

Customise your SETTINGS

After setup, open the SETTINGS sheet and update:

  • ADMIN_LOGIN — change from CHANGE_ME to your password
  • APP_TITLE — your business name
  • CFG_SALES_REP — default sales rep name
  • PRIMARY_COLOUR — your brand hex colour
  • SECTION_A_NAME … D_NAME — rename product sections
4

Deploy the Web App

Once your data is ready:

  • Click Extensions → Apps Script
  • Click Deploy → New Deployment
  • Type: Web App · Execute as: Me · Access: Anyone
  • Copy the URL into SETTINGS → WEB_APP_URL
